# Student Registration System

A responsive student registration system built with HTML, CSS, and JavaScript. It lets users add, edit, delete, and persist student records in browser local storage.

## Features

- Add student name, student ID, class, email ID, contact number, and address.
- Edit or delete existing records.
- Save records after page refresh using `localStorage`.
- Validate names, numeric student IDs, valid email addresses, and contact numbers with at least 10 digits.
- Prevent empty records from being added.
- Add a vertical scrollbar dynamically when the records list grows.
- Responsive layout for mobile, tablet, and desktop screens.

## File Structure

```text
index.html
styles.css
script.js
README.md
```

## How To Run

Open `index.html` in any modern browser.
